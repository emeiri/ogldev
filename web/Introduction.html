Introduction

The primary requirement for running these tutorials is a graphics card that supports at least OpenGL 3.3.

The tutorial code was originally developed on Ubuntu Linux using NetBeans, but it can also be built from the command line using make.

Required packages (Ubuntu)

  1. g++
  2. freeglut3-dev
  3. libglew1.5-dev

Original test environment

CPU: Intel Core 2 Extreme QX9770

OS: Ubuntu Linux 11.04 (64-bit)

Video card: GIGABYTE NVIDIA GeForce GTX550 TI

Driver version: 275.21

Please report compilation or runtime issues on other platforms. Fixes are always appreciated.
